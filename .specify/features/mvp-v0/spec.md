## Feature Summary

把"不知道送什么、不知道怎么说、不知道为什么没出单"三件事，封装成一条 10 步 AI 原生链路：「表达需求 → 选品 → 内容 → 封面 → 发布 → 订单 → 复盘 → AI 下一步 → AI 记忆」。

## User Flows

### Flow 1：表达需求 → AI 推荐（核心 happy path）

1. 用户进入「AI 对话」Tab，看到「Hi，我是小礼」的欢迎语；
2. 用户用自然语言描述：「今天准备送谁、预算多少、什么场景」；
3. 小礼解析后回写一张「需求概览卡」（对象 / 关系 / 预算 / 场景）并展示「为你推荐 X 份合适的礼物」；
4. 用户点击某份礼物，进入详情 → 进入内容生成工作台。

### Flow 2：内容生成 → 封面设计 → 发布

1. 内容生成工作台显示「文案生成 / 话术 / 短组画 / 图文策略」四个 Tab；
2. 默认选「文案生成」，显示 3 段风格化文案（分享型 / 实测型 / 情感型），用户可「复制 / 换一版 / 优化 / 更多风格」；
3. 用户点「生成封面」→ 跳到「封面设计」页面，可选 4 个模板（简约高级 / 温柔治愈 / 清新自然 / 复古轻奢）；
4. 封面确认后点「下载保存」，回到「发布确认」页面，看到文案 + 封面组合；
5. 点「确认发布」→ 显示「发布成功」页面 + 「查看发布内容」+ 「分享给好友」+ 「继续生成」三个按钮。

### Flow 3：订单佣金追踪

1. 用户进入「订单与佣金」Tab，看到「今日预计佣金 / 本月预计佣金 / 预计订单佣金」三张顶部数据卡；
2. 下方有「全部 / 全部订单 / 待结算 / 已结算 / 已提现」筛选；
3. 订单列表展示订单号、订单金额、订单状态、佣金。

### Flow 4：经营复盘（AI 分析）

1. 用户在「探索」Tab 进入「经营复盘」；
2. 小礼给出本周复盘：总发布 / 总订单 / 预计佣金 / 核心数据（点击率 1.26% / 点击转化率 3.11% / 客单价 ¥196.3）；
3. 给出「下一步建议」三条（如：「加观察期：同宣传色的礼物推荐」、「继续提高分享图文案流畅度」）；
4. 用户可点「生成下周发布计划」→ 跳到「AI 下一步计划」页面。

### Flow 5：AI 下一步计划

1. 「AI 下一步计划」页面展示「小礼为你制定了下周计划（05.12–05.18）」；
2. 列出「送礼场景 / 推荐对象 / 预算范围」三列的 4 个执行项（如：「送长辈健康礼物 母亲节 ¥200R 21:00」）；
3. 底部「不同关系给同周」说明 + 「一键生成内容」CTA。

### Flow 6：我的 AI 记忆

1. 「我的」Tab 进入 AI 记忆页面，可编辑；
2. 展示「主要推荐对象 / 常用预算区间 / 偏好文章风格 / 擅长商品类目 / 最佳发布时间 / 经营目标」6 项记忆条目；
3. 用户可点编辑修改每项，或一键「编辑我的偏好」。

## Acceptance Criteria

### AC-1 全局体验
- [ ] 全局背景 `--brand-bg` (#FAF9F6)，主色 `--brand-green` (#23B26D)。
- [ ] 品牌 Logo「礼有方 GiftPilot」+ 主口号在 footer 出现。
- [ ] 全局底部 Tab：`AI 对话 / 探索 / 创作 / 我的`。
- [ ] 6 大 AI 原生体验标签（选对礼 / 谈得好 / 做得美 / 发得准 / 看得清 / 改得好）在合适位置出现。

### AC-2 表达需求页
- [ ] 顶部问候「Hi，我是小礼」+ AI 头像；
- [ ] 欢迎副标题「今天准备送谁、预算多少、什么场景，我来帮你想得体又有新意」；
- [ ] 输入区支持自然语言 + 语音图标 + 长按语音；
- [ ] 输入后展开「需求概览卡」（对象 / 关系 / 预算 / 场景）；
- [ ] 下方有「今日推荐机会」横向滚动卡（礼物图 + 价格 + 一键生成）。

## Out of Scope（MVP v0.6 阶段不做）

- 真实微信支付（mock 订单）；
- 真实第三方商品 API（mock 商品库）；
- 真实朋友圈 / 视频号发布（生成内容 + 复制到剪贴板）；
- 礼有方 Pro / 礼有方云 / 礼有方开放平台；
- SaaS 多租户 / 机构管理 / AI 任务分发；
- 国际化（先 zh-CN）。
